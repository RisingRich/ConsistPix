# ConsistPix

ConsistPix is a collaborative multi-level fusion framework for few-shot cross-modality medical image translation. It aims to reconstruct missing target-modality images from limited paired data while preserving anatomically meaningful structures.

Our method combines structure-aware alignment, hierarchical multi-scale fusion, and spatial-frequency consistency into a unified framework to improve translation quality and structural fidelity.
